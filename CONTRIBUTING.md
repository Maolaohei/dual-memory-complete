## 📝 贡献指南

### 如何开始

1. **Fork 仓库**
   ```bash
   git clone https://github.com/YOUR_USERNAME/dual-memory-complete.git
   ```

2. **创建分支**
   ```bash
   git checkout -b feature/your-feature
   ```

3. **提交更改**
   ```bash
   git commit -m "feat: add your feature"
   ```

4. **提交 PR**
   - 描述清楚改动
   - 附上测试截图
   - 关联相关 Issue

### 代码规范

- **风格**: 使用 ESLint 标准配置
- **注释**: 复杂逻辑必须注释
- **测试**: 新功能必须带测试
- **文档**: 更新 README 和 API 文档
