## 中文长文版本

🎉 **Dual-Memory Complete 正式开源！**

一个**生产级**的 AI Agent 记忆系统，整合了向量存储、智能压缩和可视化。

### 🏗️ 双层架构

**Core v3 (核心层)**
- LanceDB 本地向量数据库
- Transformers.js 嵌入模型 (384维)
- 毫秒级实时写入
- 时间轴版本化
- 冲突检测与解决
- P0/P1/P2 三级置信衰减

**Enhanced v4 (增强层)**
- 相似记忆智能压缩
- 时间轴/统计/网络图可视化
- 完整 CRUD 主动干预
- 访问统计追踪

| 传统系统 | Dual-Memory |
|---------|-------------|
| 静默压缩丢失数据 | 显式压缩命令 + 预览模式 |
| 不知道存了什么 | 三种可视化视图 |
| 无法手动管理 | 完整 CLI 控制 |
| 相似记忆重复 | AI 智能总结 |
