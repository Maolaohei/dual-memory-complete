## 📦 新增功能

### 1. HyDE 检索优化
- **功能:** 生成假设记忆提升检索召回
- **效果:** 语义距离远的查询也能召回相关记忆
- **示例:** "用户偏好" → "用户喜欢用户..."

### 2. 时间衰减评分
- **公式:** 相似度×0.7 + 新鲜度×0.2 + 频率×0.1
- **效果:** 最近记忆优先，高频记忆加成
- **加成:** P0×1.2, P1×1.0, P2×0.8, P3×0.6

### 3. 核心文件向量化
- **文件:** SOUL.md (8切片), USER.md (2切片), AGENTS.md (5切片)
- **效果:** 按需检索，减少启动 Token
- **总计:** 15 个切片

### 4. Token 预算管理
- **预算:** 核心800 + 记忆600 + 对话1500 + 系统300 + 预留200
- **功能:** 估算、截断、预算报告
- **效果:** 防止上下文溢出

### 5. P2 记忆生命周期
- **P2:** 30天降级 → 60天归档 → 90天删除
- **P3:** 14天归档 → 30天删除
- **合并:** 相似度 > 0.92 自动合并

### 6. 检索反馈回路
- **功能:** 记录检索事件、统计命中次数、分析检索效果
- **输出:** 热门查询、高频记忆、优化建议
